安装Python

  1. 访问Python下载地址，点击下图中蓝色框的链接，像普通文件一样下载即可。

  2. 下载完毕之后双击该安装包（一般情况下在下载目录中），出现下图，勾选 Add Python 3.7 to PATH 之后点击 Customize installation。

  3. 这是安装的选择，直接点击 Next

  4. 点击 Browse 选择安装路径，推荐安装在非系统盘，下面的例子安装在E盘，你也可以根据自己的情况选择路径，然后点击Install。

  5. 这时候就会开始安装：

  6. 恭喜，安装成功！
